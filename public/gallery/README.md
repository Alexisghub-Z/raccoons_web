# Galería de Trabajos Realizados

Esta carpeta contiene imágenes adicionales de trabajos realizados que se mostrarán en la galería principal.

## Convención de Nombres

Las imágenes deben nombrarse como:
```
trabajo-1.jpg
trabajo-2.jpg
trabajo-3.jpg
...
trabajo-20.jpg
```

## Cómo Agregar Imágenes

1. Agrega tus imágenes con el formato `trabajo-[número].jpg`
2. El número puede ir del 1 al 20 (o más si expandes el componente)
3. La galería las cargará automáticamente

## Formato Recomendado

- **Formato**: JPG o PNG
- **Tamaño recomendado**: 800x1000px (vertical, relación 4:5)
- **Peso máximo**: 300KB por imagen
- **Calidad**: 80% para JPG

## Tipos de Imágenes Sugeridas

- Motos en proceso de reparación
- Detalles de trabajos realizados
- Motor, suspensión, frenos
- Personalizaciones
- Pintura y acabados
- Clientes satisfechos con sus motos

## Cantidad Actual Soportada

El MosaicGallery actual está configurado para mostrar hasta **14 imágenes únicas**.

Si actualmente solo usas las 5 imágenes del taller, puedes agregar hasta **9 imágenes más** de trabajos realizados.
